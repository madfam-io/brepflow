
# MarchingCubes Node

**Category:** Algorithmic / Geometry

Extract isosurface using marching cubes

## Parameters


### isovalue

- **Default:** 0
- **Min:** -100
- **Max:** 100



### resolution
- **Type:** number
- **Default:** 32
- **Min:** 8
- **Max:** 128



### smooth
- **Type:** boolean
- **Default:** true

### scalarField
- **Type:** Properties
- **Required:** Yes



## Outputs


### mesh
- **Type:** Shape



### vertices
- **Type:** Point[]



### normals
