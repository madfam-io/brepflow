
# DistanceField Node

**Category:** Algorithmic / Geometry

Compute signed distance field

## Parameters


### resolution
- **Type:** number
- **Default:** 50
- **Min:** 10
- **Max:** 200



### bounds
- **Type:** Vector
- **Default:** "100,100,100"


- **Description:** Bounding box size


### signed
- **Type:** boolean
- **Default:** true





## Inputs


### geometry
- **Type:** Shape
- **Required:** Yes



## Outputs


### field
- **Type:** Properties



### isosurface
- **Type:** Shape



### gradient
- **Type:** Vector[]
