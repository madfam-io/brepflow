**Category:** Transform

Wrap shape around cylinder or sphere

### type
- **Type:** enum
- **Default:** "cylinder"





### radius
- **Type:** number
- **Default:** 50
- **Min:** 0.1
- **Max:** 10000



### angle
- **Type:** number
- **Default:** 360
- **Min:** 0
- **Max:** 360



## Inputs


### shape
- **Type:** Shape
- **Required:** Yes



## Outputs


### wrapped
- **Type:** Shape
