
# Align Node

**Category:** Transform

Align shapes to each other

## Parameters


### alignX
- **Type:** enum
- **Default:** "center"





### alignY
- **Type:** enum
- **Default:** "center"





### alignZ
- **Type:** enum
- **Default:** "none"





## Inputs


### shapes
- **Type:** Shape[]
- **Required:** Yes
- **Description:** Shapes to align


### reference
- **Type:** Shape
- **Required:** No
- **Description:** Reference shape


## Outputs


### aligned
- **Type:** Shape[]
- **Description:** Aligned shapes
