
# Rotate Node

**Category:** Transform

Rotate shape around axis

## Parameters


### angle
- **Type:** number
- **Default:** 45
- **Min:** -360
- **Max:** 360
- **Description:** Rotation angle in degrees


### axisX
- **Type:** number
- **Default:** 0
- **Min:** -1
- **Max:** 1
- **Description:** Axis X component


### axisY
- **Type:** number
- **Default:** 0
- **Min:** -1
- **Max:** 1
- **Description:** Axis Y component


### axisZ
- **Type:** number
- **Default:** 1
- **Min:** -1
- **Max:** 1
- **Description:** Axis Z component


### centerX
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### centerY
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### centerZ
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### copy
- **Type:** boolean
- **Default:** true





## Inputs


### shape
- **Type:** Shape
- **Required:** Yes



## Outputs


### rotated
- **Type:** Shape
- **Description:** Rotated shape
