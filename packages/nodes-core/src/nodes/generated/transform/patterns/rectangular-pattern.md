
# RectangularPattern Node

**Category:** Transform / Patterns

Creates a 2D rectangular grid pattern

## Parameters


### countX
- **Type:** number
- **Default:** 4
- **Min:** 1
- **Max:** 100



### countY
- **Type:** number
- **Default:** 3
- **Min:** 1
- **Max:** 100



### spacingX
- **Type:** number
- **Default:** 20
- **Min:** 0.1
- **Max:** 10000



### spacingY
- **Type:** number
- **Default:** 20
- **Min:** 0.1
- **Max:** 10000



### staggered
- **Type:** boolean
- **Default:** false


- **Description:** Stagger alternate rows


## Inputs


### shape
- **Type:** Shape
- **Required:** Yes



## Outputs


### shapes
- **Type:** Shape[]



### compound
- **Type:** Shape
