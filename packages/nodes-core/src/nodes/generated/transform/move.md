
# Move Node

**Category:** Transform

Translate shape in 3D space

## Parameters


### X
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000
- **Description:** X translation


### Y
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000
- **Description:** Y translation


### Z
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000
- **Description:** Z translation


### Copy
- **Type:** boolean
- **Default:** true


- **Description:** Create copy or move original


## Inputs


### Shape
- **Type:** Shape
- **Required:** Yes
- **Description:** Shape to move


## Outputs


### Moved
- **Type:** Shape
- **Description:** Moved shape
