
# Move Node

**Category:** Transform

Translate shape in 3D space

## Parameters


### x
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000
- **Description:** X translation


### y
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000
- **Description:** Y translation


### z
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000
- **Description:** Z translation


### copy
- **Type:** boolean
- **Default:** true


- **Description:** Create copy or move original


## Inputs


### shape
- **Type:** Shape
- **Required:** Yes
- **Description:** Shape to move


## Outputs


### moved
- **Type:** Shape
- **Description:** Moved shape
