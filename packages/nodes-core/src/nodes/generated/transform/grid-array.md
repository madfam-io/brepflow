- **Max:** 100

- **Type:** number
- **Default:** 3
- **Min:** 1
- **Max:** 100



### countZ
- **Type:** number
- **Default:** 1
- **Min:** 1
- **Max:** 100



### spacingX
- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000



### spacingY
- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000



### spacingZ
- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000



### merge
- **Type:** boolean
- **Default:** false
