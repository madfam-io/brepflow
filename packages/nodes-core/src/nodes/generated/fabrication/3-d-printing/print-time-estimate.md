
# PrintTimeEstimate Node

**Category:** Fabrication / 3D Printing

- **Default:** 60
- **Min:** 10
- **Max:** 300



### Travel Speed
- **Type:** number
- **Default:** 120
- **Min:** 50
- **Max:** 500



### Layer Height
- **Type:** number
- **Default:** 0.2
- **Min:** 0.05
- **Max:** 1

### Model

- **Required:** Yes

- **Type:** Number
