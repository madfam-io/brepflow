
# Capsule Node

**Category:** Solid / Primitives

Create a capsule (cylinder with hemisphere caps)

## Parameters


### radius
- **Type:** number
- **Default:** 25
- **Min:** 0.1
- **Max:** 10000
- **Description:** Capsule radius


### height
- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000
- **Description:** Cylindrical section height


## Inputs

This node has no inputs.

## Outputs


### solid
- **Type:** Solid
- **Description:** Generated capsule
