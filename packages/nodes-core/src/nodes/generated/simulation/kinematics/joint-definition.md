
# JointDefinition Node

**Category:** Simulation / Kinematics

Define kinematic joint

## Parameters


### Joint Type
- **Type:** enum
- **Default:** "revolute"





### Axis
- **Type:** vector3
- **Default:** [0,0,1]

- **Type:** number
- **Default:** -180
- **Min:** -360
- **Max:** 360



### Max Limit
- **Type:** number
- **Default:** 180
- **Min:** -360
- **Max:** 360
