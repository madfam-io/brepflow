
# OBJExport Node

**Category:** Interoperability / Export

Export mesh to OBJ format

### Include Normals

- **Default:** true





### Include Tex Coords

- **Default:** false





### Smoothing
- **Type:** boolean
- **Default:** true

### Meshes
- **Type:** Shape[]
- **Required:** Yes



### File Path
- **Type:** string
- **Required:** Yes



## Outputs


### Success
- **Type:** boolean



### Vertex Count
- **Type:** number



### Face Count
- **Type:** number
