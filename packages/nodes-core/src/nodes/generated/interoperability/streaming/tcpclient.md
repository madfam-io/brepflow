**Category:** Interoperability / Streaming

- **Type:** string

- **Max:** 65535



### Timeout

- **Default:** 30
- **Min:** 1
- **Max:** 300

### Data
- **Type:** string
- **Required:** No



## Outputs


### Connected
- **Type:** boolean



### Response
- **Type:** string
