
# BallBearing Node

**Category:** MechanicalEngineering / Bearings

Create ball bearing assembly

## Parameters


### innerDiameter
- **Type:** number
- **Default:** 20
- **Min:** 3
- **Max:** 200
- **Description:** Bore diameter in mm


### outerDiameter
- **Type:** number
- **Default:** 47
- **Min:** 10
- **Max:** 400
- **Description:** Outer diameter in mm


### width
- **Type:** number
- **Default:** 14
- **Min:** 3
- **Max:** 100
- **Description:** Width in mm


### ballCount
- **Type:** number
- **Default:** 8
- **Min:** 5
- **Max:** 20



### showCage
- **Type:** boolean
- **Default:** true

### center

- **Required:** Yes



### axis
