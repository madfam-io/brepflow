
# SpurGear Node

**Category:** MechanicalEngineering / Gears

Create standard spur gear

## Parameters


### module
- **Type:** number
- **Default:** 2
- **Min:** 0.5
- **Max:** 20
- **Description:** Gear module in mm


### teeth
- **Type:** number
- **Default:** 20
- **Min:** 6
- **Max:** 200
- **Description:** Number of teeth


### pressureAngle
- **Type:** number
- **Default:** 20
- **Min:** 14.5
- **Max:** 25
- **Description:** Pressure angle in degrees


### width
- **Type:** number
- **Default:** 20
- **Min:** 1
- **Max:** 200
- **Description:** Face width in mm


### hubDiameter
- **Type:** number
- **Default:** 20
- **Min:** 5
- **Max:** 100
- **Description:** Hub bore diameter


## Inputs


### center
- **Type:** Point
- **Required:** No



### axis
- **Type:** Vector
- **Required:** No



## Outputs


### gear
- **Type:** Shape



### pitchCircle
- **Type:** Wire
