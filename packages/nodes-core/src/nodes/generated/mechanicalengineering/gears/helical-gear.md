
# HelicalGear Node

**Category:** MechanicalEngineering / Gears

Create helical gear with angle

## Parameters


### module
- **Type:** number
- **Default:** 2
- **Min:** 0.5
- **Max:** 20



### teeth
- **Type:** number
- **Default:** 20
- **Min:** 6
- **Max:** 200



### helixAngle
- **Type:** number
- **Default:** 15
- **Min:** 0
- **Max:** 45
- **Description:** Helix angle in degrees


### width
- **Type:** number
- **Default:** 20
- **Min:** 1
- **Max:** 200



### handedness
- **Type:** enum
- **Default:** "right"





## Inputs


### center
- **Type:** Point
- **Required:** No



## Outputs


### gear
- **Type:** Shape



### profile
- **Type:** Wire
