
# ClampingCollar Node

**Category:** MechanicalEngineering / Fasteners

Create shaft collar/clamp

## Parameters


### Shaft Diameter
- **Type:** number
- **Default:** 10
- **Min:** 3
- **Max:** 50



### Outer Diameter
- **Type:** number
- **Default:** 20
- **Min:** 8
- **Max:** 80



### Width
- **Type:** number
- **Default:** 8
- **Min:** 3
- **Max:** 20



### Clamp Type
- **Type:** enum
- **Default:** "set-screw"





## Inputs


### Position
- **Type:** Point
- **Required:** Yes



## Outputs


### Collar
- **Type:** Shape



### Bore
- **Type:** Wire
