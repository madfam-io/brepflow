
# Difference Node

**Category:** Boolean

Subtract tool shapes from base shape

## Parameters


### keepOriginals
- **Type:** boolean
- **Default:** false





### fuzzyValue
- **Type:** number
- **Default:** 1e-7
- **Min:** 0
- **Max:** 1



## Inputs


### base
- **Type:** Shape
- **Required:** Yes
- **Description:** Base shape


### tools
- **Type:** Shape[]
- **Required:** Yes
- **Description:** Shapes to subtract


## Outputs


### result
- **Type:** Shape
- **Description:** Difference result
