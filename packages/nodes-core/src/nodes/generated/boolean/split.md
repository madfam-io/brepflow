
# Split Node

**Category:** Boolean

Split shapes by each other

## Parameters


### keepAll
- **Type:** boolean
- **Default:** true


- **Description:** Keep all fragments


## Inputs


### shapes
- **Type:** Shape[]
- **Required:** Yes
- **Description:** Shapes to split


### tools
- **Type:** Shape[]
- **Required:** Yes
- **Description:** Splitting tools


## Outputs


### fragments
- **Type:** Shape[]
- **Description:** Split fragments
