
# Imprint Node

**Category:** Boolean

Imprint one shape onto another

## Parameters


### depth
- **Type:** number
- **Default:** 1
- **Min:** 0.01
- **Max:** 1000
- **Description:** Imprint depth


## Inputs


### base
- **Type:** Shape
- **Required:** Yes
- **Description:** Base shape


### imprint
- **Type:** Shape
- **Required:** Yes
- **Description:** Shape to imprint


## Outputs


### result
- **Type:** Shape
- **Description:** Imprinted shape
