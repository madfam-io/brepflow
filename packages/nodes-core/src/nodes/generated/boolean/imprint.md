
# Imprint Node

**Category:** Boolean

Imprint one shape onto another

## Parameters


### Depth
- **Type:** number
- **Default:** 1
- **Min:** 0.01
- **Max:** 1000
- **Description:** Imprint depth


## Inputs


### Base
- **Type:** Shape
- **Required:** Yes
- **Description:** Base shape


### Imprint
- **Type:** Shape
- **Required:** Yes
- **Description:** Shape to imprint


## Outputs


### Result
- **Type:** Shape
- **Description:** Imprinted shape
