
# Glue Node

**Category:** Boolean

Glue shapes together at common faces

## Parameters


### Tolerance
- **Type:** number
- **Default:** 1e-7
- **Min:** 0
- **Max:** 1



## Inputs


### Shapes
- **Type:** Shape[]
- **Required:** Yes
- **Description:** Shapes to glue


## Outputs


### Result
- **Type:** Shape
- **Description:** Glued shape
