
# Glue Node

**Category:** Boolean

Glue shapes together at common faces

## Parameters


### tolerance
- **Type:** number
- **Default:** 1e-7
- **Min:** 0
- **Max:** 1



## Inputs


### shapes
- **Type:** Shape[]
- **Required:** Yes
- **Description:** Shapes to glue


## Outputs


### result
- **Type:** Shape
- **Description:** Glued shape
