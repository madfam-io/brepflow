
# FieldHistogram Node

**Category:** Fields / Analysis

Generate histogram of field values

## Parameters


### bins
- **Type:** number
- **Default:** 20
- **Min:** 5
- **Max:** 100
- **Description:** Number of histogram bins


## Inputs


### field
- **Type:** Field
- **Required:** No



### domain
- **Type:** Geometry
- **Required:** No



## Outputs


### binCenters
- **Type:** NumberList



### binCounts
- **Type:** NumberList



### binEdges
- **Type:** NumberList
