
# FieldColorMap Node

**Category:** Fields / Visualization

Visualize field values as colors

### colorScheme

- **Default:** "\"viridis\""


- **Description:** Color scheme for visualization


### minValue
- **Type:** number
- **Default:** 0


- **Description:** Minimum field value


### maxValue
- **Type:** number
- **Default:** 1


- **Description:** Maximum field value

- **Required:** Yes
