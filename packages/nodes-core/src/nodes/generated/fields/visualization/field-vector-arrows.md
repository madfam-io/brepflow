
# FieldVectorArrows Node

**Category:** Fields / Visualization

Display vector field as arrows

## Parameters


### arrowScale
- **Type:** number
- **Default:** 1
- **Min:** 0.1
- **Max:** 10
- **Description:** Scale factor for arrows


### density
- **Type:** number
- **Default:** 0.5
- **Min:** 0
- **Max:** 1
- **Description:** Display density (0-1)


## Inputs


### field
- **Type:** VectorField
- **Required:** No



### domain
- **Type:** Geometry
- **Required:** Yes



## Outputs


### arrows
- **Type:** GeometrySet
