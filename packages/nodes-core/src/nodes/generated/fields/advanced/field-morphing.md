
# FieldMorphing Node

**Category:** Fields / Advanced

Morph between two fields

## Parameters


### factor
- **Type:** number
- **Default:** 0.5
- **Min:** 0
- **Max:** 1
- **Description:** Morphing factor (0=field1, 1=field2)


### interpolation
- **Type:** enum
- **Default:** "\"linear\""


- **Description:** Interpolation method


## Inputs


### field1
- **Type:** Field
- **Required:** No



### field2
- **Type:** Field
- **Required:** No



## Outputs


### morphedField
- **Type:** Field
