
# Sweep Node

**Category:** Advanced / Sweep

Sweep profile along path

## Parameters


### twistAngle
- **Type:** number
- **Default:** 0
- **Min:** -360
- **Max:** 360
- **Description:** Twist along path


### scaleFactor
- **Type:** number
- **Default:** 1
- **Min:** 0.01
- **Max:** 100
- **Description:** Scale at end


### keepOrientation
- **Type:** boolean
- **Default:** false





### solid
- **Type:** boolean
- **Default:** true


- **Description:** Create solid or surface


## Inputs


### profile
- **Type:** Wire
- **Required:** Yes



### path
- **Type:** Wire
- **Required:** Yes



### auxiliarySpine
- **Type:** Wire
- **Required:** No



## Outputs


### shape
- **Type:** Shape
