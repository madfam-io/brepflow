
# Draft Node

**Category:** Advanced / Draft

Add draft angle to faces

## Parameters


### angle
- **Type:** number
- **Default:** 3
- **Min:** -30
- **Max:** 30
- **Description:** Draft angle in degrees


### pullDirection
- **Type:** vector3
- **Default:** [0,0,1]





### neutralPlane
- **Type:** vector3
- **Default:** [0,0,0]





## Inputs


### solid
- **Type:** Shape
- **Required:** Yes



### facesToDraft
- **Type:** Face[]
- **Required:** Yes



## Outputs


### drafted
- **Type:** Shape
