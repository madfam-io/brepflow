
# Jog Node

**Category:** SheetMetal / Bends

Create jog offset in sheet

## Parameters


### jogOffset
- **Type:** number
- **Default:** 10
- **Min:** 0.1
- **Max:** 1000



### jogAngle
- **Type:** number
- **Default:** 90
- **Min:** 0
- **Max:** 180



### bendRadius
- **Type:** number
- **Default:** 3
- **Min:** 0.1
- **Max:** 100



## Inputs


### sheet
- **Type:** Shape
- **Required:** Yes



### jogLine
- **Type:** Edge
- **Required:** Yes



## Outputs


### result
- **Type:** Shape
