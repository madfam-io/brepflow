
# Jog Node

**Category:** SheetMetal / Bends

Create jog offset in sheet

## Parameters


### Jog Offset
- **Type:** number
- **Default:** 10
- **Min:** 0.1
- **Max:** 1000



### Jog Angle
- **Type:** number
- **Default:** 90
- **Min:** 0
- **Max:** 180



### Bend Radius
- **Type:** number
- **Default:** 3
- **Min:** 0.1
- **Max:** 100



## Inputs


### Sheet
- **Type:** Shape
- **Required:** Yes



### Jog Line
- **Type:** Edge
- **Required:** Yes



## Outputs


### Result
- **Type:** Shape
