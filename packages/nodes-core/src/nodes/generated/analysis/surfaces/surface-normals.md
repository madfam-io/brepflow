
# SurfaceNormals Node

**Category:** Analysis / Surfaces

Calculate surface normal vectors

## Parameters


### Density
- **Type:** number
- **Default:** 20
- **Min:** 5
- **Max:** 100



### Vector Length
- **Type:** number
- **Default:** 5
- **Min:** 1
- **Max:** 50



### Show Vectors
- **Type:** boolean
- **Default:** true





## Inputs


### Surface
- **Type:** Face
- **Required:** Yes



## Outputs


### Normal Vectors
- **Type:** Vector[]



### Normal Lines
- **Type:** Wire[]



### Sample Points
- **Type:** Point[]
