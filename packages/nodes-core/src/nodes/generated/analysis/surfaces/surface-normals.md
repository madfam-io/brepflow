
# SurfaceNormals Node

**Category:** Analysis / Surfaces

Calculate surface normal vectors

## Parameters


### density
- **Type:** number
- **Default:** 20
- **Min:** 5
- **Max:** 100



### vectorLength
- **Type:** number
- **Default:** 5
- **Min:** 1
- **Max:** 50



### showVectors
- **Type:** boolean
- **Default:** true





## Inputs


### surface
- **Type:** Face
- **Required:** Yes



## Outputs


### normalVectors
- **Type:** Vector[]



### normalLines
- **Type:** Wire[]



### samplePoints
- **Type:** Point[]
