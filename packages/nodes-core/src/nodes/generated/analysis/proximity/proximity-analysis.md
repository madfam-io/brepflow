
# ProximityAnalysis Node

**Category:** Analysis / Proximity

Analyze proximity between multiple objects

## Parameters


### threshold
- **Type:** number
- **Default:** 1
- **Min:** 0.1
- **Max:** 100



### showConnections
- **Type:** boolean
- **Default:** true





## Inputs


### objects
- **Type:** Shape[]
- **Required:** Yes



## Outputs


### proximityPairs
- **Type:** Shape[][]



### distances
- **Type:** number[]



### connections
- **Type:** Wire[]
