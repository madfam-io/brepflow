
# BoundingBox Node

**Category:** Analysis / Properties

Get bounding box of shape

## Parameters

This node has no parameters.

## Inputs


### shape
- **Type:** Shape
- **Required:** Yes



## Outputs


### min
- **Type:** Point
- **Description:** Minimum corner


### max
- **Type:** Point
- **Description:** Maximum corner


### center
- **Type:** Point
- **Description:** Box center


### dimensions
- **Type:** Vector
- **Description:** Box dimensions (width, depth, height)
