
# Circle Node

**Category:** Sketch / Basic

Create a circle

## Parameters


### centerX
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### centerY
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### centerZ
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### radius
- **Type:** number
- **Default:** 50
- **Min:** 0.1
- **Max:** 10000



### filled
- **Type:** boolean
- **Default:** true


- **Description:** Create as face (filled) or wire (outline)


## Inputs

This node has no inputs.

## Outputs


### shape
- **Type:** Shape
- **Description:** Circle face or wire
