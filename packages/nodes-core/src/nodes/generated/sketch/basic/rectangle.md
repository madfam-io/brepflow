
# Rectangle Node

**Category:** Sketch / Basic

Create a rectangle

## Parameters


### centerX
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### centerY
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### width
- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000



### height
- **Type:** number
- **Default:** 50
- **Min:** 0.1
- **Max:** 10000



### filled
- **Type:** boolean
- **Default:** true


- **Description:** Create as face (filled) or wire (outline)


### cornerRadius
- **Type:** number
- **Default:** 0
- **Min:** 0
- **Max:** 1000
- **Description:** Corner rounding radius


## Inputs

This node has no inputs.

## Outputs


### shape
- **Type:** Shape
- **Description:** Rectangle face or wire
