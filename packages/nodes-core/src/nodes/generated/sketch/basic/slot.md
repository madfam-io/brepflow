
# Slot Node

**Category:** Sketch / Basic

Create a slot (rounded rectangle)

## Parameters


### centerX
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### centerY
- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000



### length
- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000



### width
- **Type:** number
- **Default:** 20
- **Min:** 0.1
- **Max:** 10000



### angle
- **Type:** number
- **Default:** 0
- **Min:** -180
- **Max:** 180
- **Description:** Rotation angle


## Inputs

This node has no inputs.

## Outputs


### face
- **Type:** Face
- **Description:** Slot face
